3. Click the element picker (top left)

## üé® CUSTOMIZE THE GRAPHICS

### To Change Pick Details:

Open `pick-preview.html` and edit these lines:

```html
<div class="game">Lakers vs Nuggets</div>  <!-- Change teams -->
<div class="pick">Lakers -3.5</div>         <!-- Change pick -->
<div class="odds">-110</div>                <!-- Change odds -->
<div class="stat-value">83%</div>           <!-- Change confidence -->
<div class="stat-value">+5.2%</div>         <!-- Change edge -->
```

### To Change Win Results:

Open `win-celebration.html` and edit:

```html
<div class="pick">Magic +4.5</div>          <!-- Change winning pick -->
<div class="stat-value green">48</div>      <!-- Change wins -->
<div class="stat-value">21</div>            <!-- Change losses -->
<div class="stat-value green">+$1,324</div> <!-- Change profit -->
```

### To Change Stats:

Open `stats-graphic.html` and edit:

```html
<div class="main-value">+$1,247</div>       <!-- Main profit number -->
<div class="stat-value green">47</div>      <!-- Wins -->
<div class="stat-value">21</div>            <!-- Losses -->
<div class="stat-value purple">69%</div>    <!-- Win rate -->
```
