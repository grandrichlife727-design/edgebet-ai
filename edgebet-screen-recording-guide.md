# EdgeBet AI - Screen Recording Guide

## Required Footage (Record in 1080x1920 Portrait)

### Shot 1: App Launch (0:18-0:23)
**Duration:** 5 seconds
**Action:**
1. Tap app icon
2. Let splash screen play (⚡ EdgeBet AI)
3. Stop when main interface loads

---

### Shot 2: Agent Animation (0:23-0:28)
**Duration:** 5 seconds
**Action:**
1. Tap "Scan for Value" button
2. Let all 7 agent icons animate in
3. Wait for scanning to complete
4. Show the picks appearing

**Gesture:** Slow, deliberate tap

---

### Shot 3: Pick Card Reveal (0:28-0:35)
**Duration:** 7 seconds
**Action:**
1. Tap on first pick card
2. Let it expand to show analysis
3. Scroll down slightly to show model breakdown
4. Tap "Analysis" button to expand

**Timing:**
- 0s: Tap card
- 2s: Card expands
- 5s: Scroll down
- 7s: Tap Analysis

---

### Shot 4: Adding to Parlay (0:35-0:40)
**Duration:** 5 seconds
**Action:**
1. Tap "+ Parlay" button on 2-3 different picks
2. Watch them slide into parlay
3. Tap parlay tab to show builder

---

### Shot 5: Telegram Notification (0:40-0:45)
**Duration:** 5 seconds
**Action:**
1. Pull down notification center

3. Tap notification to open app
4. Show the pick that was delivered

### Shot 6: Line Movement Graph (0:45-0:52)

## Recording Settings

### iOS
```
Settings → Control Center → Screen Recording (add if not present)
Settings → Camera → Record Video → 1080p HD at 30fps
```

**To Record:**
1. Swipe down from top-right corner
2. Long-press record button
3. Enable microphone (optional)
4. Tap Start Recording
5. 3-second countdown

### Android
```
Settings → Display → Screen Recorder
Resolution: 1080p
Frame rate: 30fps
Touch indicators: ON
```

**To Record:**
1. Swipe down twice for Quick Settings
2. Tap Screen Recorder
3. Choose audio source
4. Tap Start

---

## Best Practices

### Before Recording
- [ ] Enable Do Not Disturb (hide notifications)
- [ ] Close all other apps
- [ ] Full battery or plugged in
- [ ] Clean phone screen
- [ ] Test recording first

### During Recording
- [ ] Use slow, deliberate gestures
- [ ] Keep fingers visible
- [ ] Pause between actions
- [ ] Record in well-lit room
- [ ] Hold phone steady

### Common Mistakes
- ❌ Recording in landscape
- ❌ Too fast gestures
- ❌ Notifications popping up
- ❌ Recording in low light
- ❌ Forgetting to enable Do Not Disturb

---

## Alternative: Screen Mirroring

If you want higher quality or easier recording:

### iOS → Mac
1. Connect iPhone to Mac with cable
2. Open QuickTime Player
3. File → New Movie Recording
4. Click dropdown next to record button → Select iPhone
5. Record in QuickTime (higher quality)

### Android → Computer
1. Enable USB debugging
2. Use scrcpy: https://github.com/Genymobile/scrcpy
3. Or use Android Studio emulator

---

## Post-Recording Checklist

- [ ] Video is 1080x1920 (portrait)
- [ ] Frame rate is 30fps
- [ ] No notifications visible
- [ ] Touch gestures are clear
- [ ] Audio is clear (if using voiceover)
- [ ] File size is reasonable (< 500MB)
