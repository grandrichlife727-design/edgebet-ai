# EdgeBet AI - Video Editing Quick Start

## Option 1: CapCut (Recommended - Free)

### Download & Install
- Desktop: https://www.capcut.com/tools/desktop-video-editor
- Mobile: App Store / Google Play

### Import Project Template
1. Open CapCut Desktop
2. Click **New Project**
3. Set resolution: **1080 x 1920** (9:16)
4. Import the files:
   - Voiceover: `voiceover-edgebet-promo.mp3`
   - Screen recordings (7 shots)
   - Agent icons (7 images)
   - Logo frames (3 images)

### Timeline Setup
```
Track 7: Text overlays (CTA, URL)
Track 6: Logo & Agent icons
Track 5: Screen recordings
Track 4: Effects & transitions
Track 3: Background elements
Track 2: Music & sound effects
Track 1: Voiceover (sync anchor)
```

### Quick Edit Steps

**Step 1: Sync Audio (5 min)**
1. Drag voiceover to Track 1
2. Mark these timestamps with markers:
   - 0:08 - Beat drop
   - 0:18 - Logo reveal
   - 0:28 - App demo starts
   - 0:52 - CTA

**Step 2: Add Video (10 min)**
1. Drag screen recordings to Track 5
2. Rough cut to match voiceover timing
3. Don't worry about transitions yet

**Step 3: Add Graphics (10 min)**
1. Add logo at 0:18 (zoom in animation)
2. Add 7 agent icons at 0:23 (staggered appearance)
3. Add text overlays for key phrases

**Step 4: Polish (15 min)**
1. Add transitions between scenes
2. Color grade (lift shadows, add cyan tint)
3. Add glow effects to logo
4. Sync cuts to music beats

**Step 5: Export (5 min)**
1. Click Export
2. Resolution: 1080x1920
3. FPS: 30
4. Format: MP4
5. Quality: High

## Option 2: Canva (Easiest)

### For Simple Promo Videos
1. Go to https://www.canva.com
2. Search "Mobile Video" template
3. Choose 9:16 aspect ratio
4. Use their stock footage + your screen recordings
5. Add text animations
6. Export as MP4

**Pros:** Drag & drop, templates, fast
**Cons:** Less control, watermarks on free tier

## Option 3: Adobe Premiere Pro

### For Professional Quality
1. Create new project
2. Sequence settings: 1080x1920, 30fps
3. Import all assets
4. Use multicam sync for audio
5. Apply Lumetri color grading
6. Add Motion Graphics templates
7. Export via Media Encoder

**Pros:** Professional tools, precise control

## Color Grading Preset

### For CapCut
1. Select all clips
2. Click **Adjust**
3. Apply these settings:
   - Brightness: +5
   - Contrast: +10
   - Saturation: +5
   - Shadows: +15
   - Highlights: -10
   - Temperature: -5 (cooler)
   - Tint: +5 (magenta)

### For Premiere Pro
Save as LUT:
```
Input LUT: None
Basic Correction:
  - Input: 0
  - Exposure: +0.5
  - Contrast: +15
  - Highlights: -20
  - Shadows: +30
  - Whites: -10
  - Blacks: -5
Creative:
  - Vibrance: +20
  - Saturation: +10
Curve:
  - Lift shadows slightly
  - Add S-curve for contrast
Color Wheel:
  - Shadows: Add cyan
  - Highlights: Add violet
```

## Text Animation Templates

### Hook Text (0:00-0:08)
- Font: Bebas Neue or Impact
- Size: 72px
- Animation: Typewriter
- Color: White
- Glow: Cyan, radius 20

### Feature Labels (0:28-0:45)
- Font: DM Sans Bold
- Size: 48px
- Animation: Slide up from bottom
- Color: Cyan (#22d3ee)
- Background: Semi-transparent black

### CTA Text (0:52-1:00)
- Font: Bebas Neue
- Size: 96px
- Animation: Scale up + pulse
- Color: White with cyan glow
- Add underline animation

## A/B Testing Your Video

Create 3 versions and test:

**Version A:** Aggressive hook
- "Stop losing money to sportsbooks"
- Fast cuts
- Dark, intense music

**Version B:** Curious hook  
- "What if you knew where the sharp money was going?"
- Slower, mysterious
- Electronic build-up

**Version C:** Social proof hook
- "Join 10,000+ bettors who found their edge"
- Testimonials featured
- Upbeat, positive music

Post all 3, see which performs best, then scale that one.

## Post-Production Checklist

Before exporting:
- [ ] Audio levels balanced (voice -6dB, music -12dB)
- [ ] No black frames at start/end
- [ ] Text is readable on mobile
- [ ] Logo is visible and clear
- [ ] URL is shown clearly
- [ ] Captions are added
- [ ] Thumbnail is created
- [ ] File name includes platform
- [ ] Watch on actual phone before posting
