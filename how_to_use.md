# How to Use This Kit

## Quick Start (2 minutes)

1. **Open the Figma file** (`Pickleball_Brand_Kit.fig`)
2. **Duplicate to your drafts** (File → Save to drafts)
3. **Start customizing:**
   - Change colors in the "Color System" page
   - Edit logo text (double-click, type your name)
   - Swap photos in mockups (drag & drop)

## What's Included

### 4 Logo Variants
- **Horizontal** - Best for websites, letterheads
- **Vertical** - Great for social profiles, business cards
- **Icon Only** - Perfect for app icons, watermarks
- **Alternate** - Vintage style for special campaigns

### Social Media Templates
All sized correctly:

- TikTok Frame: 1080 × 1920 px

### Print Mockups
- T-shirt (front/back)
- Tote bag
- Sticker sheet

## Customization Tips

**Changing Colors:**
1. Go to "Color System" page
2. Click any color swatch
3. Edit the HEX code
4. All linked elements update automatically

**Editing Text:**
1. Double-click any text layer
2. Type your team/club name
3. Press Enter

**Exporting for Print:**
1. Select the mockup frame
2. Right-click → Export
3. Choose PNG, 2x scale
4. Upload to Printful/Printify
